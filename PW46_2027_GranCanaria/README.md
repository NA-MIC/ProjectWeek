# Welcome to the web page for the 46th Project Week!

[This event](https://projectweek.na-mic.org/PW46_2027_GranCanaria/README.html) will take place January 25th - 29th, 2027 in Las Palmas de Gran Canaria, Spain, in person. More details will be posted on this page as the event approaches. If you have any questions, you can contact the [organizers](#organizers).
